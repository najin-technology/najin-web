# 옛 네이버 블로그 → 공식 사이트 마이그레이션 가이드

## 배경

- 구글에서 "나진테크" 검색 시 옛 네이버 블로그(`blog.naver.com/...`)가 1등으로 노출되고 공식 사이트(`https://www.najin-tech.com`)는 아래에 위치.
- 이는 옛 블로그가 오랜 기간 인덱싱되어 도메인 권위·백링크가 누적된 결과. 공식 사이트는 신규 도메인(<1년)이라 권위가 낮음.
- **목표**: 옛 블로그의 권위·트래픽을 공식 사이트로 점진적 전이.

## 핵심 원칙

1. **옛 블로그를 갑자기 비공개·삭제하지 말 것.** 검색 노출이 사라져 유입이 끊기고, 백링크도 소실됨.
2. **옛 글을 그대로 두되, 공식 사이트로의 링크를 강하게 박을 것.** 신호를 점진적으로 옮긴다.
3. **새 글은 공식 사이트 우선 발행.** 블로그에는 요약·링크만.

---

## 단계별 액션

### 1단계 — 사이드바·프로필 (1일)

**전체 글에 자동으로 노출되는 위치에 신 사이트 링크.**

- **블로그 프로필** 소개란:
  ```
  나진테크 공식 홈페이지 → https://www.najin-tech.com
  본 블로그는 아카이브이며, 최신 정보는 공식 사이트에서 확인하세요.
  ```
- **사이드바 위젯** 또는 **프롤로그(메인)**: 공식 사이트 배너 + 링크.
- **블로그 타이틀/소개글**에도 공식 사이트 URL 명시.

→ 이렇게 하면 모든 글에서 공식 사이트로 백링크가 자동 생성됨 (블로그 글 N개 × 1링크).

### 2단계 — 기존 글 상단에 안내문 (1~2주)

**중요한 글부터** 차근차근. 한 번에 다 하지 말고 트래픽 높은 순으로.

각 글 본문 **맨 위**에 다음 박스 추가:

```
═══════════════════════════════════
👉 본 글은 아카이브입니다.
   최신 본문은 공식 사이트에서 보실 수 있습니다:
   https://www.najin-tech.com/ko/posts/<slug>
   (해당 글에 매칭되는 공식 사이트 글로 deep-link)
═══════════════════════════════════
```

- 매칭되는 공식 사이트 글이 있으면 해당 URL로, 없으면 `/ko/portfolio` 또는 `/ko/business`.
- 매주 5~10개씩 점진 추가 (한 번에 다 바꾸면 spammy 시그널로 인식될 수 있음).

### 3단계 — 본문 축약 + 공식 사이트로 유도 (1~2개월)

기존 글 본문이 길다면 **앞부분 50%만 남기고** 뒷부분에:

```
…(중략)…

📖 전문 보기: https://www.najin-tech.com/ko/posts/<slug>
   - 고화질 사진
   - 추가 사양·재료 정보
   - 견적 문의 폼 직접 연결
```

- 콘텐츠 가치를 공식 사이트에 집중시켜, 검색 사용자가 "더 보려면 공식 사이트로" 자연스럽게 이동.

### 4단계 — 신규 글 정책 변경 (즉시)

**오늘부터** 적용:

1. 신규 글은 **공식 사이트의 `/ko/posts/<slug>` 또는 `/ko/notices/<id>`** 에 먼저 작성.
2. 블로그에는 **요약(150~300자) + "전문 보기" 링크만**.
3. 블로그용 썸네일은 동일하게 사용해도 무방.

```
[블로그 글 템플릿]
제목: {공식 사이트와 동일한 제목}

본문:
{공식 사이트 첫 단락 150~300자 그대로 복사}

전문 보기 → https://www.najin-tech.com/ko/posts/<slug>

#나진테크 #양산 #우레탄성형 (관련 태그)
```

### 5단계 — 백링크 지속 빌드 (지속)

공식 사이트의 새 글이 발행되면, 블로그에도 동일 주제로 짧은 안내글(요약 + 링크)을 같이 올림. → 블로그가 "공식 사이트의 announcement 채널" 역할.

## 모니터링

매주 1회 확인:

| 지표 | 도구 | 목표 |
|---|---|---|
| 블로그 → 공식 사이트 클릭 수 | 네이버 애널리틱스 (블로그 통계) | 주당 N건 (점진 증가) |
| 공식 사이트 referrer 중 blog.naver.com | Vercel Analytics 또는 GSC | 트래픽 비중 ↑ |
| 검색 결과 순위 | "나진테크" 구글·네이버 직접 검색 | 4주 후: 첫 페이지 진입 |

---

## ⚠️ 하지 말 것

- 옛 블로그를 **비공개**·**삭제** 처리 (검색 자산 소멸).
- **모든 글 본문에 동시에** 동일 문구 삽입 (스팸 시그널).
- **자동 리다이렉트**: 네이버 블로그는 외부 도메인 301/302 리다이렉트 미지원. 시도 불가.
- 글마다 **link spamming** (한 글에 공식 사이트 링크 10개 이상): 자연스럽게 1~2개만.

---

## 예상 타임라인

- **Day 1~3**: 사이드바·프로필 정비 (가장 큰 1회 효과)
- **Week 1~4**: 트래픽 상위 글부터 안내문 + 본문 축약 (주 5~10개)
- **Week 4 이후**: 신규 글은 공식 우선, 블로그는 요약만
- **Month 2~3**: 검색 결과에서 공식 사이트가 블로그를 추월하기 시작
- **Month 4~6**: "나진테크" 쿼리에서 공식 사이트가 1등으로 안정 진입

도메인 권위 빌드는 시간이 걸립니다. 4주 내 효과가 안 보여도 정상.
