# 백링크·디렉터리 등록 체크리스트

## 배경

신규 도메인(<1년)인 `najin-tech.com`은 도메인 권위(authority)가 낮습니다. 검색엔진이 "이 사이트 믿어도 되나?" 판단할 때 가장 큰 시그널이 **외부 사이트의 링크(backlinks)**.

이 문서는 **공식·반공식 디렉터리 + 산업 협회 + B2B 포털**에 우선 등록할 곳을 우선순위 순으로 정리.

## 작업 원칙

- **품질 > 수량.** 한국 산업 디렉터리 10개가 잡지 백링크 100개보다 가치 큼.
- **NAP 일관성.** Name(나진테크), Address(경남 양산시 산막공단남14길 170), Phone(055-367-2596)을 모든 등록처에 **완전히 동일하게** 입력. 작은 차이도 검색엔진에 별개 entity로 인식될 수 있음.
- **공식 사이트 URL** 은 항상 `https://www.najin-tech.com` (www. 포함, https).
- **정기 점검.** 등록 사이트가 폐쇄되거나 정보가 노후화되지 않았는지 분기 1회 확인.

## P3 — 위키·공공 (장기 권위, 1~3개월)

### 위키데이터 (Wikidata, https://www.wikidata.org)
- **가장 중요한 GEO 자산 중 하나.**
- "NAJIN TECHNOLOGY" entity 생성 → Q번호 부여
- LLM(ChatGPT, Perplexity 등)이 학습/실시간 검색 시 인용 가능성 ↑
- 등록 정보: 영문/한글 사명, 설립일, 본사 위치, 산업 분류, 공식 사이트 URL, ISO 인증 등
- ⚠️ Wikidata는 비영리 검증 커뮤니티가 운영함. 광고성 표현 금지, 출처 인용 필수.

### 영문 위키피디아 (검토)
- 회사 규모(직원 50명) + 25년 + 인증 + 거래처 수준이면 notability 기준 통과 가능성 있음
- 단, 자체 작성하면 conflict-of-interest 으로 삭제될 수 있음
- 권장: 신뢰할 수 있는 외부 매체 보도(연합뉴스, 경상일보, 자동차산업 매체 등)가 누적된 후 시도.

### 나무위키 (검토, 신중히)
- 한국에서 검색 노출 효과 큼
- 가이드라인이 까다로워 자작 시 삭제 가능성. 외부 보도 후 자연 등재 유도.

- **12개월 후**: "나진테크" 쿼리에서 공식 사이트 1페이지 안정 진입
