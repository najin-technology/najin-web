# 콘텐츠 운영 캘린더

## 배경

검색엔진은 **신선한 콘텐츠를 정기적으로 발행하는 사이트**를 권위 있는 사이트로 인식합니다. 신규 도메인일수록 "정기 발행" 시그널이 도메인 권위 빌드에 핵심.

또한 longtail 키워드(예: "양산 우레탄 부품 견적", "EV 모터 합성수지 가공")로 유입되는 사용자는 검색 의도가 명확해 견적 전환율이 높습니다.

## 발행 빈도

**월 4건** 권장. 주 1건 페이스.

너무 자주 발행하면 품질이 떨어지고, 너무 드물면 신선도 시그널이 약해짐.

---

## 4건 구성 (월 단위)

### 1. 기술 사례 1건 — `/posts/<slug>`

**가장 SEO 가치가 큰 콘텐츠.** 특정 부품·재료·공정의 실제 작업 사례.

- 제목 패턴: "[거래처/산업] [부품/공정] 가공 사례 — [핵심 키워드]"
  - 예: "현대모비스 변속기 케이스 우레탄 부싱 가공 사례 — 정밀도 ±0.05mm"
- 구성:
  1. 한 줄 요약 (160자 이내, meta description 활용)
  2. 어떤 문제를 풀었나 (고객 요구사항)
  3. 어떤 재료·공정 선택했나
  4. 결과 사진 3~5장 (`<Image>` 컴포넌트 활용, alt 필수)
  5. 거래처 정보 (선택, /clients/[slug] 와 deep-link)
  6. 견적 CTA
- 길이: 본문 1500~3000자
- 태그: 우레탄/합성수지/CNC/금형/EV 중 해당 + 거래처명

### 2. 인증·수상·뉴스 1건 — `/notices/<id>`

회사 공식 발표.

- 제목 예: "ISO 9001 갱신 인증 획득 (2026)" / "정부 우수기업 선정" / "신규 거래처 추가 — XX社"
- 구성:
  1. 발표 요약
  2. 의미·배경 (왜 이번이 중요한가)
  3. 향후 계획
- 길이: 500~1000자

### 3. 산업 트렌드 1건 — `/posts/<slug>`

거래처 관계 없이 **업계 동향 분석**. longtail 키워드 흡수용.

- 제목 패턴: "[산업/기술] 트렌드와 [회사 강점] 관점"
  - 예: "EV 배터리팩 부품 가공 동향 — 합성수지 활용 사례"
  - 예: "친환경 우레탄 소재의 등장과 산업 적용 가능성"
- 구성: 시장 동향 + 회사가 어떻게 대응 중인가 (자연스러운 회사 노출)
- 길이: 1000~2000자

### 4. 시즈널 1건 — `/notices/<id>` 또는 `/posts/<slug>`

명절·휴무·전시회·견학 등 일회성 이벤트.

- 제목 예: "2026 추석 연휴 휴무 안내" / "K-Auto Show 2026 참가 안내" / "공장 견학 프로그램 운영"
- 구성: 짧고 명확하게. 200~500자.

## 월간 발행 일정 템플릿

```
1주차 (월요일): 기술 사례 1건  ← 가장 중요, 미리 사진·자료 준비
2주차 (월요일): 인증/뉴스 1건
3주차 (월요일): 산업 트렌드 1건
4주차 (월요일): 시즈널 / 잡담 1건
```

월요일 발행 권장: 검색엔진 크롤러가 주말 변경분을 수집하기 시작하는 시점.

## 키워드 사전 (longtail 30선)

각 글 작성 시 다음 키워드 중 1~2개를 자연스럽게 본문에 포함.

### 우레탄
- 양산 우레탄 성형
- 우레탄 금형베이스
- 우레탄 롤러 제작
- TANK PAD 우레탄
- 산업용 우레탄 부싱

### 합성수지
- PE 환봉 가공
- POM 정밀가공
- 엔지니어링 플라스틱 절삭
- 합성수지 부품 제작 양산
- PP 시트 가공

### CNC
- CNC 정밀가공 견적
- MCT 3D 형상 가공
- CNC 시제품 가공
- 자동차 변속기 부품 가공
- 정밀 부품 외주 가공

### 금형
- 우레탄 금형 제작
- 산업용 금형 설계
- 양산 금형 제작 업체
- 시제품 금형

### EV
- EV 배터리팩 부품
- 전기차 모터 부품 가공
- EV 합성수지 부품
- 친환경 모빌리티 부품 제조

### 회사 일반
- 양산 정밀 가공
- 경남 자동차 부품 제조
- 25년 정밀 가공 업체
- 현대자동차 협력사 가공
- ISO 9001 부품 제조

## 작성 체크리스트

각 글 발행 전:

- [ ] 제목에 longtail 키워드 1개 자연스럽게 포함 (60자 이내)
- [ ] 첫 단락에 핵심 키워드 + 회사명 노출 (160자 이내, meta description 활용)
- [ ] 이미지 3장 이상, 모든 이미지에 `alt` 텍스트
- [ ] 한국어 본문 + (가능하면) 영어 번역 본문 (`title_en`, `content_en`)
- [ ] 관련 페이지로 내부 링크 2~3개 (`/business`, `/portfolio`, `/clients/<slug>`)
- [ ] 견적 CTA 1개 이상
- [ ] 태그 2~5개

## 효과 측정

월말 1회 GSC 확인:

- **노출수**: 신규 글이 검색결과에 노출된 횟수
- **클릭수**: 노출 → 클릭 비율 (CTR)
- **평균 순위**: 어떤 키워드에서 몇 위?
- **인기 페이지**: 어떤 글이 트래픽을 가장 많이 모으나? → 다음 달 비슷한 패턴 반복

목표:
- **3개월**: 월 노출수 1,000+ 도달
- **6개월**: 월 노출수 5,000+ 도달, longtail 키워드 10개 이상에서 1페이지 진입
- **12개월**: 월 노출수 20,000+, 핵심 키워드(우레탄 성형, CNC 정밀가공 등)에서 1페이지 진입

## 주의사항

- **저품질 양산 금지.** 4건 발행이 어려우면 2~3건만 발행하되 품질 유지. 1년 묵혀둔 글이 매주 새 글보다 가치 있음.
- **기존 글 업데이트도 발행으로 카운트 가능.** "2024년 기준" → "2026년 기준" 으로 갱신하고 `updated_at` 변경하면 검색엔진은 신선도 시그널로 인식.
- **AI 자동 생성 콘텐츠 주의.** Google E-E-A-T 가이드라인상 가치 없는 AI 글은 색인 제외 가능. 회사 실제 작업 경험을 담은 1차 자료(사진, 사양서, 거래처 멘트) 가 핵심.
