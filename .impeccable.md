# Impeccable Design Context — NAJIN TECHNOLOGY

## Design Context

### Users
B2B 제조업 발주처 담당자 (40~60대 중심). 네이버 검색 → 블로그 → 웹사이트로 유입. "여기 맡겨도 되겠다"는 신뢰가 핵심 판단 기준. 도면을 가지고 견적을 요청하려는 사람들. 전화보다 온라인 견적 폼으로 먼저 파악하고 싶음.

### Brand Personality
**신뢰 · 정밀 · 기술력**
- 24년 역사의 제조업체 — 가볍거나 트렌디한 느낌 X
- 국내외 주요 완성차·배터리·소재 기업 납품 실적이 증명하는 기술력
- ISO 9001, CLEAN 사업장, 특허 보유 — 인증으로 뒷받침되는 신뢰
- 실제 가공 사진이 가장 강력한 전문성 증거

### Aesthetic Direction
**Industrial Refined** — 공장의 견고함 + 현대적 세련됨
- 어둡고 무거운 제조업 느낌이 아닌, 밝고 신뢰감 있는 톤
- 실제 제품/장비 사진이 디자인의 중심 (스톡 이미지 X)
- 데이터와 실적이 눈에 보이는 구조 (연혁, 거래처, 인증)
- 어르신들이 편하게 볼 수 있는 가독성 (작은 글씨 X)

### Anti-References
- 너무 화려한 스타트업 느낌 (그라데이션, 네온, 글래스모피즘)
- 너무 미니멀해서 정보가 없는 느낌
- 일본식 과밀 텍스트 레이아웃
- 중국 알리바바 스타일 과도한 배너/팝업

### Design Principles
1. **사진이 곧 신뢰** — 실제 가공품, 장비, 공장 사진이 최고의 설득 도구
2. **증명으로 말한다** — 거래처 로고, 인증 배지, 연혁이 빈 약속보다 강력
3. **어르신 친화적** — 큰 글씨, 명확한 동선, 복잡하지 않은 레이아웃
4. **행동 유도** — 모든 페이지에서 견적 문의로의 명확한 경로
5. **견고한 전문성** — 가볍지 않고, 무겁지 않은. 정밀 가공업체다운 절제된 힘

### Color Direction
- Primary: Navy (#1B2A4A) — 신뢰, 전문성
- Accent: Copper (#B87333) — 금속, 정밀, 따뜻한 포인트
- Surfaces: Warm grays — 차가운 테크 느낌이 아닌 따뜻한 제조업
- 로고 블루 (#00428B) — 회사 로고에서 가져온 브랜드 컬러

### Typography
- Korean: Noto Sans KR — 한국 제조업 표준, 가독성 우수
- English/UI: Geist — 현대적이면서 기술적
- 본문 최소 16px, 모바일에서도 14px 이하 금지
